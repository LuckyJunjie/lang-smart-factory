## 仪表盘功能

| 功能 | 说明 |
|------|------|
| 项目列表 | 左侧边栏显示所有项目 |
| 实时状态 | 显示活跃任务、进度、最后更新时间 |
| 历史时间线 | 展示每个步骤的执行状态和时间 |
| 统计卡片 | 活跃项目、总任务、通过率、进行中 |

## 访问方式

- 完整仪表盘: `/dashboard/full`
- 状态 API: `/api/v1/dashboard/stats`
